# NotePlus 应用说明文档

## 应用概述

NotePlus是一款功能强大的笔记记录应用，专为高效记录和管理个人想法、事件和任务而设计。通过项目管理、时间记录、标签系统等功能，帮助用户更好地整理和追踪日常活动。

## 主要功能

### 笔记记录
- **快速记录**：通过简洁的输入界面，快速记录当下的想法和活动
- **可扩展输入框**：点击输入框旁边的扩展按钮，可切换单行/多行输入模式
- **富文本支持**：支持文本格式化，便于结构化记录内容

### 项目管理
- **多项目支持**：可创建多个独立项目，分类管理不同类型的笔记
- **项目切换**：通过顶部工具栏的项目按钮快速切换项目
- **项目回收站**：删除的项目会移至回收站，防止意外删除
- **项目重命名**：可以为项目重新命名，更好地组织内容

### 时间管理
- **时间标记**：为每条记录添加精确的时间戳
- **时间区间**：可设置开始和结束时间，记录活动持续时间
- **时间筛选**：通过时间区间筛选和查找记录

### 标签系统
- **自定义标签**：创建个性化标签，包括名称和颜色
- **多标签关联**：为单条记录添加多个标签
- **标签筛选**：通过标签快速找到相关记录

### 数据管理
- **导出功能**：支持导出为CSV或JSON格式，便于数据备份和分析
- **长按删除**：长按记录可以删除，并有确认提示防止误操作
- **数据备份**：支持数据导出和恢复

## 使用指南

### 基本操作
1. **添加记录**：在顶部输入框中输入内容，点击"保存"按钮
2. **扩展输入框**：点击输入框旁的扩展按钮，切换到多行输入模式
3. **设置时间区间**：点击"开始"和"结束"时间选择器，为记录设置时间范围
4. **添加标签**：点击"添加标签"按钮，从现有标签中选择或创建新标签
5. **删除记录**：长按列表中的记录，在确认对话框中选择"删除"

### 项目管理
1. **切换项目**：点击顶部工具栏的项目按钮，从下拉菜单中选择项目
2. **创建项目**：在项目管理菜单中选择"创建新项目"，输入名称确认
3. **管理项目**：在项目管理中可以重命名、删除项目
4. **回收站操作**：在项目管理菜单中进入回收站，可以恢复或永久删除项目

### 数据导出
1. **导出数据**：点击工具栏的导出按钮
2. **选择格式**：选择CSV或JSON格式
3. **查看导出文件**：导出完成后可直接打开文件查看

## 技术特性

- 使用SQLite数据库存储记录和标签数据
- 支持Android 10+的存储权限处理
- 多线程数据处理，确保UI响应流畅
- 数据库事务处理，保证数据完整性
