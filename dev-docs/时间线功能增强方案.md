# 时间线功能增强方案

## 一、需求概述

对现有的时间线功能进行增强，主要包括：
1. **扩展时间范围选项**：在原有"最近一周"的基础上，增加"2周"、"1个月"、"3个月"等选项，默认仍为"最近一周"
2. **增加排序方向配置**：允许用户选择"逆序"（最新的在前）或"顺序"（最旧的在前）

## 二、现有实现分析

### 2.1 核心类结构

#### TimeRangeFilter 枚举
- 位置：`app/src/main/java/person/notfresh/noteplus/core/TimeRangeFilter.java`
- 现有选项：
  - `LAST_DAY(1)` - 最近1天
  - `LAST_WEEK(7)` - 最近7天
  - `LAST_MONTH(30)` - 最近30天

#### GlobalTimeline 类
- 位置：`app/src/main/java/person/notfresh/noteplus/core/GlobalTimeline.java`
- 核心方法：
  - `loadGlobalTimeline(TimeRangeFilter timeRange, boolean descending)` - 支持时间范围和排序方向
  - `loadGlobalTimelineLastWeek(boolean descending)` - 便捷方法，固定一周范围

#### MainActivity 中的调用
- 位置：`app/src/main/java/person/notfresh/noteplus/MainActivity.java` (约2188行)
- 当前实现：硬编码调用 `loadGlobalTimelineLastWeek(true)`，固定为最近一周、逆序

#### UI布局
- 位置：`app/src/main/res/layout/dialog_timeline.xml`
- 当前结构：标题栏（含项目数量）+ 列表视图 + 关闭按钮

## 三、设计方案

### 3.1 数据层扩展

#### 3.1.1 扩展 TimeRangeFilter 枚举

**需要新增的枚举值：**
```java
LAST_TWO_WEEKS(14),      // 最近2周
LAST_THREE_MONTHS(90);   // 最近3个月（按90天计算）
```

**建议的完整枚举结构：**
- `LAST_DAY(1)` - 最近1天
- `LAST_WEEK(7)` - 最近1周（默认）
- `LAST_TWO_WEEKS(14)` - 最近2周
- `LAST_MONTH(30)` - 最近1个月
- `LAST_THREE_MONTHS(90)` - 最近3个月

**说明：**
- 保持现有枚举值不变，确保向后兼容
- 新增枚举值遵循相同的构造模式（天数）

#### 3.1.2 GlobalTimeline 类扩展

**需要新增的便捷方法（可选）：**
```java
public List<Comment> loadGlobalTimelineLastTwoWeeks(boolean descending)
public List<Comment> loadGlobalTimelineLastThreeMonths(boolean descending)
```

**说明：**
- 这些方法不是必需的，因为 `loadGlobalTimeline(TimeRangeFilter, boolean)` 已经足够通用
- 如果添加，主要是为了保持API一致性

### 3.3 逻辑层实现

#### 3.3.1 MainActivity 修改点

**修改方法：** `showTimelineDialog()`

**主要修改内容：**

1. **初始化UI控件**
   ```java
   // 获取新增的UI控件
   Spinner rangeSpinner = dialogView.findViewById(R.id.timelineRangeSpinner);
   Switch sortSwitch = dialogView.findViewById(R.id.timelineSortSwitch);
   ```

2. **设置时间范围选项**
   ```java
   // 创建时间范围选项数组
   String[] rangeOptions = {"最近1周", "最近2周", "最近1个月", "最近3个月"};
   ArrayAdapter<String> rangeAdapter = new ArrayAdapter<>(this, 
       android.R.layout.simple_spinner_item, rangeOptions);
   rangeAdapter.setDropDownViewResource(android.R.layout.simple_spinner_dropdown_item);
   rangeSpinner.setAdapter(rangeAdapter);
   rangeSpinner.setSelection(0); // 默认选择"最近1周"
   ```

3. **设置排序方向**
   ```java
   // 默认逆序（最新的在前）
   sortSwitch.setChecked(true);
   ```

4. **创建数据加载方法（提取为独立方法）**
   ```java
   private void loadTimelineData(ListView listView, TextView countText, 
                                  TimeRangeFilter range, boolean descending) {
       ProgressDialog progressDialog = new ProgressDialog(this);
       progressDialog.setMessage("正在加载时间线...");
       progressDialog.setCancelable(false);
       progressDialog.show();
       
       new Thread(() -> {
           try {
               GlobalTimeline globalTimeline = new GlobalTimeline(projectManager);
               List<Comment> timelineItems = globalTimeline.loadGlobalTimeline(range, descending);
               
               runOnUiThread(() -> {
                   progressDialog.dismiss();
                   countText.setText(timelineItems.size() + " 项");
                   TimelineAdapter adapter = new TimelineAdapter(timelineItems);
                   listView.setAdapter(adapter);
                   // ... 设置点击事件等
               });
           } catch (Exception e) {
               // ... 错误处理
           }
       }).start();
   }
   ```

5. **监听选择变化并重新加载数据**
   ```java
   // 监听时间范围变化
   rangeSpinner.setOnItemSelectedListener(new AdapterView.OnItemSelectedListener() {
       @Override
       public void onItemSelected(AdapterView<?> parent, View view, int position, long id) {
           TimeRangeFilter selectedRange = getTimeRangeFromPosition(position);
           boolean descending = sortSwitch.isChecked();
           loadTimelineData(timelineListView, itemCountText, selectedRange, descending);
       }
       
       @Override
       public void onNothingSelected(AdapterView<?> parent) {}
   });
   
   // 监听排序方向变化
   sortSwitch.setOnCheckedChangeListener((buttonView, isChecked) -> {
       int selectedPosition = rangeSpinner.getSelectedItemPosition();
       TimeRangeFilter selectedRange = getTimeRangeFromPosition(selectedPosition);
       loadTimelineData(timelineListView, itemCountText, selectedRange, isChecked);
   });
   ```

6. **辅助方法：位置到枚举的映射**
   ```java
   private TimeRangeFilter getTimeRangeFromPosition(int position) {
       switch (position) {
           case 0: return TimeRangeFilter.LAST_WEEK;      // 最近1周
           case 1: return TimeRangeFilter.LAST_TWO_WEEKS; // 最近2周
           case 2: return TimeRangeFilter.LAST_MONTH;     // 最近1个月
           case 3: return TimeRangeFilter.LAST_THREE_MONTHS; // 最近3个月
           default: return TimeRangeFilter.LAST_WEEK;
       }
   }
   ```

7. **初始加载**
   ```java
   // 初始加载默认数据（最近1周，逆序）
   loadTimelineData(timelineListView, itemCountText, 
                    TimeRangeFilter.LAST_WEEK, true);
   ```

### 3.4 可选：用户偏好保存

**如果需要记住用户的选择：**

1. **使用SharedPreferences保存用户偏好**
   ```java
   private static final String PREFS_TIMELINE = "timeline_prefs";
   private static final String KEY_RANGE = "time_range";
   private static final String KEY_SORT = "sort_descending";
   
   // 保存偏好
   private void saveTimelinePreferences(int rangePosition, boolean descending) {
       SharedPreferences prefs = getSharedPreferences(PREFS_TIMELINE, MODE_PRIVATE);
       prefs.edit()
           .putInt(KEY_RANGE, rangePosition)
           .putBoolean(KEY_SORT, descending)
           .apply();
   }
   
   // 读取偏好
   private void loadTimelinePreferences() {
       SharedPreferences prefs = getSharedPreferences(PREFS_TIMELINE, MODE_PRIVATE);
       int rangePosition = prefs.getInt(KEY_RANGE, 0); // 默认最近1周
       boolean descending = prefs.getBoolean(KEY_SORT, true); // 默认逆序
       // 应用到UI
   }
   ```

2. **在用户选择变化时保存**
   ```java
   rangeSpinner.setOnItemSelectedListener(...) {
       // ... 加载数据
       saveTimelinePreferences(position, sortSwitch.isChecked());
   }
   ```

   - 时间范围越大，数据量可能越大，注意加载性能
   - 可以考虑添加加载进度提示
   - 如果数据量很大，可以考虑分页加载

3. **用户体验**
   - 切换选项时应该有明确的加载提示
   - 可以考虑添加"刷新"按钮，手动触发重新加载
   - 如果加载时间较长，可以考虑添加取消功能

4. **UI适配**
   - 确保新增的UI控件在不同屏幕尺寸下正常显示
   - 考虑横屏和竖屏的布局适配

5. **错误处理**
   - 确保在数据加载失败时有友好的错误提示
   - 处理边界情况（如没有数据时）
