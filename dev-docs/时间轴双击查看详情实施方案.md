# 时间轴双击查看详情实施方案

## 一、需求概述

为时间轴功能添加双击查看详情功能：
1. **双击时间轴中的内容项**，可以查看对应的Note详情（包含该Note的所有Comment）
2. **如果是Comment项**，双击后查看关联的Note详情
3. **如果是Note项**，双击后查看该Note的详情（包含所有Comment）

## 二、现有实现分析

### 2.1 时间轴数据结构

#### Comment对象结构
- `id`: Comment的ID
- `noteId`: 关联的Note ID
- `itemType`: 类型（NOTE、COMMENT、DATE_DIVIDER）
- `projectName`: 所属项目名称
- `content`: 内容
- `timestamp`: 时间戳
- `cost`: 花费金额

#### 关键方法
- `Comment.getItemType()`: 获取项类型
- `Comment.getNoteId()`: 获取关联的Note ID
- `Comment.getProjectName()`: 获取项目名称
- `Comment.isNote()`: 判断是否为Note转换来的

### 2.2 现有功能

#### 项目切换
- `ProjectContextManager.switchToProject(String projectName)`: 切换项目
- `MainActivity.switchProject(String projectName)`: 切换项目并刷新UI

#### 数据加载
- `NoteDbHelper.getCommentsForNote(long noteId)`: 获取Note的所有Comment
- `NoteDbHelper.loadNotes()`: 加载Note列表
- `NoteDataLoader.loadNoteWithComments()`: 加载Note及其所有Comment

#### UI交互
- 当前时间轴列表使用 `ListView.setOnItemClickListener()` 处理点击
- 已有TODO注释：需要添加跳转到Note详情的逻辑

### 2.3 需要实现的功能

1. **双击检测机制**：区分单击和双击
2. **项目切换逻辑**：如果Note属于不同项目，需要切换项目
3. **Note详情对话框**：显示Note内容和所有Comment
4. **数据加载**：从对应项目加载Note及其所有Comment

## 三、设计方案

### 3.1 双击检测方案

**方案A：使用Handler延迟检测（推荐）**
- 优点：实现简单，兼容性好
- 缺点：需要处理单击延迟

**方案B：使用GestureDetector**
- 优点：更精确的双击检测
- 缺点：需要额外的依赖

**推荐使用方案A**，通过Handler延迟执行单击操作，在延迟期间如果再次点击则视为双击。

### 3.2 Note详情对话框设计

## 五、技术实现细节

### 5.1 双击检测实现

```java
// 双击检测相关变量
private Handler clickHandler = new Handler();
private Runnable clickRunnable;
private long lastClickTime = 0;
private static final long DOUBLE_CLICK_DELAY = 300; // 双击间隔时间（毫秒）

// 在ListView的点击事件中
listView.setOnItemClickListener((parent, view, position, id) -> {
    Comment item = timelineItems.get(position);
    
    // 跳过日期分割线
    if (item.getItemType() == TimelineItemType.DATE_DIVIDER) {
        return;
    }
    
    long currentTime = System.currentTimeMillis();
    
    // 如果距离上次点击时间小于双击间隔，认为是双击
    if (currentTime - lastClickTime < DOUBLE_CLICK_DELAY) {
        // 取消单击延迟执行
        clickHandler.removeCallbacks(clickRunnable);
        
        // 执行双击操作
        handleDoubleClick(item);
        
        lastClickTime = 0; // 重置
    } else {
        // 记录点击时间
        lastClickTime = currentTime;
        
        // 延迟执行单击操作（如果在这期间没有双击）
        clickRunnable = () -> {
            // 单击操作（如果需要）
            handleSingleClick(item);
            lastClickTime = 0;
        };
        clickHandler.postDelayed(clickRunnable, DOUBLE_CLICK_DELAY);
    }
});
```

### 5.2 项目切换和详情加载

```java
private void handleDoubleClick(Comment item) {
    String targetProjectName = item.getProjectName();
    long noteId = item.getNoteId();
    
    // 检查是否需要切换项目
    String currentProject = projectManager.getCurrentProject();
    if (!currentProject.equals(targetProjectName)) {
        // 需要切换项目
        switchProjectAndShowDetail(targetProjectName, noteId);
    } else {
        // 直接显示详情
        showNoteDetailDialog(noteId, targetProjectName);
    }
}

private void switchProjectAndShowDetail(String projectName, long noteId) {
    // 关闭时间轴对话框
```

### 5.3 Note详情数据加载

```java
private void loadNoteDetail(long noteId, String projectName) {
    ProgressDialog progressDialog = new ProgressDialog(this);
    progressDialog.setMessage("正在加载详情...");
    progressDialog.setCancelable(false);
    progressDialog.show();
    
    new Thread(() -> {
        try {
            // 获取对应项目的数据库Helper
            NoteDbHelper dbHelper = projectManager.getDbHelperForProject(projectName);
            if (dbHelper == null) {
                throw new Exception("项目不存在");
            }
            
            // 加载Note基本信息
            Note note = loadNoteById(dbHelper, noteId);
            if (note == null) {
                throw new Exception("Note不存在");
            }
            
            // 加载Note的所有Comment
            List<Comment> comments = loadCommentsForNote(dbHelper, noteId);
            
            // 加载Note的标签（如果有）
            List<Tag> tags = loadTagsForNote(dbHelper, noteId);
            
            // 返回UI线程显示
            runOnUiThread(() -> {
                progressDialog.dismiss();
                showNoteDetailDialog(note, comments, tags);
            });
        } catch (Exception e) {
            runOnUiThread(() -> {
                progressDialog.dismiss();
                Toast.makeText(this, "加载失败: " + e.getMessage(), Toast.LENGTH_SHORT).show();
            });
        }
    }).start();
}
```
