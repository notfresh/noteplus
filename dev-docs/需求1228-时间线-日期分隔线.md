方案二：在数据源中插入日期标记项

实现思路：
在创建 TimelineAdapter 前，遍历 timelineItems，在跨天处插入日期标记对象
创建日期标记类（如 DateDivider），或使用 Comment 的特殊状态标识

修改 TimelineAdapter：
getView() 判断 item 类型

日期标记返回日期分割线布局（新建 item_timeline_date_divider.xml）

普通 Comment 返回原布局

优点：
逻辑清晰，数据与视图分离
分割线独立，样式灵活

缺点：
需要修改数据源结构
需要处理两种 item 类型

getItem() 和点击事件需要适配
