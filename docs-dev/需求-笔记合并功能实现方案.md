# 笔记合并功能实现方案

## 功能需求

在笔记长按菜单中添加"合并到..."选项，允许将当前笔记合并到另一个笔记中作为追加内容（Comment）。

## 实现步骤

### 1. 修改笔记操作菜单

**位置**：`MainActivity.showNoteOptionsMenu()`

**修改内容**：
- 在菜单选项中添加"合并到..."选项
- 菜单项顺序：`{"复制到剪切板", "追加内容", "合并到...", isPinned ? "取消置顶" : "置顶", "删除"}`

**代码示例**：
```java
private void showNoteOptionsMenu(View anchorView, long noteId) {
    AlertDialog.Builder builder = new AlertDialog.Builder(this);
    builder.setTitle("选择操作");
    
    // 检查当前置顶状态
    boolean isPinned = dbHelper.isNotePinned(noteId);
    String[] options = {
        "复制到剪切板", 
        "追加内容", 
        "合并到...",  // 新增选项
        isPinned ? "取消置顶" : "置顶", 
        "删除"
    };
    
    builder.setItems(options, (dialog, which) -> {
        switch (which) {
            case 0: // 复制到剪切板
                copyToClipboard(noteId);
                break;
            case 1: // 添加追加
                showAddCommentDialog(noteId, null);
                break;
            case 2: // 合并到...（新增）
                showMergeToDialog(noteId);
                break;
            case 3: // 置顶/取消置顶
                togglePinNote(noteId);
                break;
            case 4: // 删除
                showDeleteConfirmDialog(noteId);
                break;
        }
    });
    
    // ... 其余代码保持不变
}
```

### 2. 创建合并目标选择对话框

**方法名**：`showMergeToDialog(long sourceNoteId)`

**功能**：
1. 获取源笔记的时间戳
2. 查询所有其他笔记（排除源笔记本身）
3. 计算每个笔记与源笔记的时间差绝对值
4. 按时间差绝对值排序（从小到大）
5. 显示前N个最接近的笔记（建议：前10-15个）
6. 用户选择后执行合并操作

**实现逻辑**：

```java
/**
 * 显示合并目标选择对话框
 * @param sourceNoteId 要合并的源笔记ID
 */
private void showMergeToDialog(long sourceNoteId) {
    // 1. 获取源笔记的时间戳
    SQLiteDatabase db = dbHelper.getReadableDatabase();
    Cursor sourceCursor = db.query(
        NoteDbHelper.TABLE_NOTES,
        new String[]{"_id", NoteDbHelper.COLUMN_TIMESTAMP},
        "_id = ?",
        new String[]{String.valueOf(sourceNoteId)},
        null, null, null
    );
    
    if (sourceCursor == null || !sourceCursor.moveToFirst()) {
        Toast.makeText(this, "无法获取笔记信息", Toast.LENGTH_SHORT).show();
        if (sourceCursor != null) sourceCursor.close();
        return;
    }
    
    long sourceTimestamp = sourceCursor.getLong(
        sourceCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_TIMESTAMP)
    );
    sourceCursor.close();
    
    // 2. 查询所有其他笔记（排除源笔记）
    Cursor allNotesCursor = db.query(
        NoteDbHelper.TABLE_NOTES,
        new String[]{"_id", NoteDbHelper.COLUMN_CONTENT, NoteDbHelper.COLUMN_TIMESTAMP},
        "_id != ?",  // 排除源笔记
        new String[]{String.valueOf(sourceNoteId)},
        null, null,
        null  // 先不排序，后面按时间差排序
    );
    
    if (allNotesCursor == null || allNotesCursor.getCount() == 0) {
        Toast.makeText(this, "没有可合并的目标笔记", Toast.LENGTH_SHORT).show();
        if (allNotesCursor != null) allNotesCursor.close();
        return;
    }
    
    // 3. 计算时间差并排序
    List<MergeTarget> mergeTargets = new ArrayList<>();
    int idIndex = allNotesCursor.getColumnIndexOrThrow("_id");
    int contentIndex = allNotesCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_CONTENT);
    int timestampIndex = allNotesCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_TIMESTAMP);
    
    while (allNotesCursor.moveToNext()) {
        long targetId = allNotesCursor.getLong(idIndex);
        String content = allNotesCursor.getString(contentIndex);
        long targetTimestamp = allNotesCursor.getLong(timestampIndex);
        
        // 计算时间差绝对值
        long timeDiff = Math.abs(targetTimestamp - sourceTimestamp);
        
        // 截取内容预览（最多50字符）
        String contentPreview = content.length() > 50 
            ? content.substring(0, 50) + "..." 
            : content;
        
        mergeTargets.add(new MergeTarget(targetId, contentPreview, targetTimestamp, timeDiff));
    }
    allNotesCursor.close();
    
    // 4. 按时间差绝对值排序（从小到大）
    mergeTargets.sort((a, b) -> Long.compare(a.timeDiff, b.timeDiff));
    
    // 5. 取前N个（建议10-15个）
    int maxTargets = Math.min(15, mergeTargets.size());
    List<MergeTarget> topTargets = mergeTargets.subList(0, maxTargets);
    
    // 6. 构建显示列表
    String[] items = new String[topTargets.size()];
    long[] targetIds = new long[topTargets.size()];
    
    for (int i = 0; i < topTargets.size(); i++) {
        MergeTarget target = topTargets.get(i);
        targetIds[i] = target.noteId;
        
        // 格式化时间差显示
        String timeDiffText = formatTimeDifference(target.timeDiff);
        
        // 格式化时间显示
        String timeText = formatTimestamp(target.timestamp);
        
        // 显示格式：时间差 | 时间 | 内容预览
        items[i] = timeDiffText + " | " + timeText + " | " + target.contentPreview;
    }
    
    // 7. 显示选择对话框
    AlertDialog.Builder builder = new AlertDialog.Builder(this);
    builder.setTitle("选择合并目标（按时间接近度排序）");
    
    builder.setItems(items, (dialog, which) -> {
        long targetNoteId = targetIds[which];
        showMergeConfirmationDialog(sourceNoteId, targetNoteId);
    });
    
    builder.setNegativeButton("取消", null);
    builder.show();
}

/**
 * 合并目标数据类（内部类）
 */
private static class MergeTarget {
    long noteId;
    String contentPreview;
    long timestamp;
    long timeDiff;  // 时间差绝对值（毫秒）
    
    MergeTarget(long noteId, String contentPreview, long timestamp, long timeDiff) {
        this.noteId = noteId;
        this.contentPreview = contentPreview;
        this.timestamp = timestamp;
        this.timeDiff = timeDiff;
    }
}

/**
 * 格式化时间差显示
 * @param timeDiffMillis 时间差（毫秒）
 * @return 格式化后的字符串，如 "5分钟前"、"2小时前"、"3天前"
 */
private String formatTimeDifference(long timeDiffMillis) {
    long seconds = timeDiffMillis / 1000;
    long minutes = seconds / 60;
    long hours = minutes / 60;
    long days = hours / 24;
    
    if (days > 0) {
        return days + "天";
    } else if (hours > 0) {
        return hours + "小时";
    } else if (minutes > 0) {
        return minutes + "分钟";
    } else {
        return seconds + "秒";
    }
}
```

### 3. 创建合并确认对话框

**方法名**：`showMergeConfirmationDialog(long sourceNoteId, long targetNoteId)`

**功能**：
- 显示合并确认信息
- 用户确认后执行合并操作

**实现逻辑**：

```java
/**
 * 显示合并确认对话框
 * @param sourceNoteId 源笔记ID
 * @param targetNoteId 目标笔记ID
 */
private void showMergeConfirmationDialog(long sourceNoteId, long targetNoteId) {
    // 获取两个笔记的内容预览
    String sourceContent = getNoteContentById(sourceNoteId);
    String targetContent = getNoteContentById(targetNoteId);
    
    String sourcePreview = sourceContent != null && sourceContent.length() > 30
        ? sourceContent.substring(0, 30) + "..."
        : (sourceContent != null ? sourceContent : "");
    
    String targetPreview = targetContent != null && targetContent.length() > 30
        ? targetContent.substring(0, 30) + "..."
        : (targetContent != null ? targetContent : "");
    
    AlertDialog.Builder builder = new AlertDialog.Builder(this);
    builder.setTitle("确认合并");
    builder.setMessage(
        "确定要将以下笔记合并到目标笔记中吗？\n\n" +
        "源笔记：\n" + sourcePreview + "\n\n" +
        "目标笔记：\n" + targetPreview + "\n\n" +
        "合并后，源笔记将被删除，其内容将作为目标笔记的追加内容。"
    );
    
    builder.setPositiveButton("确认合并", (dialog, which) -> {
        performMerge(sourceNoteId, targetNoteId);
    });
    
    builder.setNegativeButton("取消", null);
    builder.show();
}
```

### 4. 执行合并操作

**方法名**：`performMerge(long sourceNoteId, long targetNoteId)`

**功能**：
1. 从数据库加载源笔记和目标笔记（包括评论）
2. 创建 Note 对象
3. 调用 `targetNote.mergeNoteAsComment(sourceNote)`
4. 将合并后的评论保存到数据库
5. 删除源笔记
6. 刷新列表

**实现逻辑**：

```java
/**
 * 执行合并操作
 * @param sourceNoteId 源笔记ID
 * @param targetNoteId 目标笔记ID
 */
private void performMerge(long sourceNoteId, long targetNoteId) {
    // 显示进度对话框
    ProgressDialog progressDialog = new ProgressDialog(this);
    progressDialog.setMessage("正在合并笔记...");
    progressDialog.setCancelable(false);
    progressDialog.show();
    
    // 在后台线程执行合并操作
    new Thread(() -> {
        try {
            SQLiteDatabase db = dbHelper.getWritableDatabase();
            
            // 1. 加载源笔记（包括评论）
            Note sourceNote = loadNoteWithComments(db, sourceNoteId);
            if (sourceNote == null) {
                runOnUiThread(() -> {
                    progressDialog.dismiss();
                    Toast.makeText(this, "无法加载源笔记", Toast.LENGTH_SHORT).show();
                });
                return;
            }
            
            // 2. 加载目标笔记（包括评论）
            Note targetNote = loadNoteWithComments(db, targetNoteId);
            if (targetNote == null) {
                runOnUiThread(() -> {
                    progressDialog.dismiss();
                    Toast.makeText(this, "无法加载目标笔记", Toast.LENGTH_SHORT).show();
                });
                return;
            }
            
            // 3. 执行合并（在内存中）
            targetNote.mergeNoteAsComment(sourceNote);
            
            // 4. 开始事务
            db.beginTransaction();
            try {
                // 5. 保存合并后的评论到数据库
                for (Comment comment : targetNote.getComments()) {
                    // 检查评论是否已存在（通过 commentId）
                    Cursor existingComment = db.query(
                        NoteDbHelper.TABLE_NOTE_COMMENTS,
                        new String[]{NoteDbHelper.COLUMN_COMMENT_ID},
                        NoteDbHelper.COLUMN_COMMENT_ID + " = ?",
                        new String[]{String.valueOf(comment.getId())},
                        null, null, null
                    );
                    
                    if (existingComment == null || existingComment.getCount() == 0) {
                        // 评论不存在，插入新评论
                        ContentValues commentValues = new ContentValues();
                        commentValues.put(NoteDbHelper.COLUMN_COMMENT_NOTE_ID, targetNoteId);
                        if (comment.getParentCommentId() != null) {
                            commentValues.put(NoteDbHelper.COLUMN_PARENT_COMMENT_ID, comment.getParentCommentId());
                        }
                        commentValues.put(NoteDbHelper.COLUMN_COMMENT_CONTENT, comment.getContent());
                        commentValues.put(NoteDbHelper.COLUMN_COMMENT_TIMESTAMP, comment.getTimestamp());
                        commentValues.put(NoteDbHelper.COLUMN_COMMENT_COST, comment.getCost());
                        
                        db.insert(NoteDbHelper.TABLE_NOTE_COMMENTS, null, commentValues);
                    }
                    
                    if (existingComment != null) {
                        existingComment.close();
                    }
                }
                
                // 6. 删除源笔记（会级联删除相关数据）
                deleteNoteInTransaction(db, sourceNoteId);
                
                // 7. 提交事务
                db.setTransactionSuccessful();
                
                // 8. 返回UI线程更新界面
                runOnUiThread(() -> {
                    progressDialog.dismiss();
                    Toast.makeText(this, "合并成功", Toast.LENGTH_SHORT).show();
                    
                    // 保存滚动位置并刷新列表
                    int firstVisiblePosition = momentsListView.getFirstVisiblePosition();
                    View firstVisibleView = momentsListView.getChildAt(0);
                    int scrollOffset = 0;
                    if (firstVisibleView != null) {
                        scrollOffset = firstVisibleView.getTop();
                    }
                    
                    // 重新查询并刷新
                    String orderBy = NoteDbHelper.COLUMN_IS_PINNED + " DESC, " + 
                            (timeDescOrder ? 
                            NoteDbHelper.COLUMN_TIMESTAMP + " DESC" : 
                            NoteDbHelper.COLUMN_TIMESTAMP + " ASC");
                    
                    Cursor newCursor = db.query(
                            NoteDbHelper.TABLE_NOTES,
                            new String[]{"_id", NoteDbHelper.COLUMN_CONTENT, NoteDbHelper.COLUMN_TIMESTAMP, NoteDbHelper.COLUMN_COST, NoteDbHelper.COLUMN_IS_PINNED},
                            null, null, null, null,
                            orderBy
                    );
                    
                    if (noteCursorWrapper != null) {
                        noteCursorWrapper.setCursor(newCursor);
                        if (noteListAdapter != null) {
                            noteListAdapter.notifyDataSetChanged();
                            if (firstVisiblePosition >= 0) {
                                int adjustedPosition = firstVisiblePosition;
                                if (adjustedPosition >= noteListAdapter.getCount()) {
                                    adjustedPosition = Math.max(0, noteListAdapter.getCount() - 1);
                                }
                                momentsListView.setSelectionFromTop(adjustedPosition, scrollOffset);
                            }
                        }
                    }
                });
                
            } finally {
                db.endTransaction();
            }
            
        } catch (Exception e) {
            e.printStackTrace();
            runOnUiThread(() -> {
                progressDialog.dismiss();
                Toast.makeText(this, "合并失败: " + e.getMessage(), Toast.LENGTH_SHORT).show();
            });
        }
    }).start();
}

/**
 * 从数据库加载笔记及其评论
 * @param db 数据库连接
 * @param noteId 笔记ID
 * @return Note 对象，包含所有评论
 */
private Note loadNoteWithComments(SQLiteDatabase db, long noteId) {
    // 1. 加载笔记基本信息
    Cursor noteCursor = db.query(
        NoteDbHelper.TABLE_NOTES,
        new String[]{
            "_id", 
            NoteDbHelper.COLUMN_CONTENT, 
            NoteDbHelper.COLUMN_TIMESTAMP, 
            NoteDbHelper.COLUMN_COST, 
            NoteDbHelper.COLUMN_IS_PINNED
        },
        "_id = ?",
        new String[]{String.valueOf(noteId)},
        null, null, null
    );
    
    if (noteCursor == null || !noteCursor.moveToFirst()) {
        if (noteCursor != null) noteCursor.close();
        return null;
    }
    
    Note note = new Note(
        noteCursor.getLong(noteCursor.getColumnIndexOrThrow("_id")),
        noteCursor.getString(noteCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_CONTENT)),
        noteCursor.getLong(noteCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_TIMESTAMP)),
        noteCursor.getDouble(noteCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_COST)),
        noteCursor.getInt(noteCursor.getColumnIndex(NoteDbHelper.COLUMN_IS_PINNED)) == 1,
        projectManager.getCurrentProject()
    );
    noteCursor.close();
    
    // 2. 加载评论
    Cursor commentsCursor = dbHelper.getCommentsForNote(noteId);
    if (commentsCursor != null && commentsCursor.getCount() > 0) {
        List<Comment> comments = new ArrayList<>();
        while (commentsCursor.moveToNext()) {
            long commentId = commentsCursor.getLong(
                commentsCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_COMMENT_ID)
            );
            int parentIdIndex = commentsCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_PARENT_COMMENT_ID);
            String content = commentsCursor.getString(
                commentsCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_COMMENT_CONTENT)
            );
            long timestamp = commentsCursor.getLong(
                commentsCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_COMMENT_TIMESTAMP)
            );
            double cost = commentsCursor.getDouble(
                commentsCursor.getColumnIndexOrThrow(NoteDbHelper.COLUMN_COMMENT_COST)
            );
            
            Long parentId = null;
            if (!commentsCursor.isNull(parentIdIndex)) {
                parentId = commentsCursor.getLong(parentIdIndex);
            }
            
            Comment comment = new Comment(
                commentId,
                noteId,
                parentId,
                content,
                timestamp,
                cost,
                projectManager.getCurrentProject(),
                TimelineItemType.COMMENT
            );
            comments.add(comment);
        }
        note.setComments(comments);
        commentsCursor.close();
    }
    
    return note;
}

/**
 * 在事务中删除笔记（不刷新UI）
 * @param db 数据库连接
 * @param noteId 笔记ID
 */
private void deleteNoteInTransaction(SQLiteDatabase db, long noteId) {
    // 1. 删除相关的标签关联
    db.delete(
        NoteDbHelper.TABLE_NOTE_TAGS,
        NoteDbHelper.COLUMN_RECORD_ID + " = ?",
        new String[]{String.valueOf(noteId)}
    );
    
    // 2. 删除相关的时间范围
    db.delete(
        NoteDbHelper.TABLE_TIME_RANGES,
        NoteDbHelper.COLUMN_NOTE_ID + " = ?",
        new String[]{String.valueOf(noteId)}
    );
    
    // 3. 删除相关的评论
    db.delete(
        NoteDbHelper.TABLE_NOTE_COMMENTS,
        NoteDbHelper.COLUMN_COMMENT_NOTE_ID + " = ?",
        new String[]{String.valueOf(noteId)}
    );
    
    // 4. 删除记录本身
    db.delete(
        NoteDbHelper.TABLE_NOTES,
        "_id = ?",
        new String[]{String.valueOf(noteId)}
    );
}
```

## 实现要点

### 1. 时间差计算
- 使用 `Math.abs(targetTimestamp - sourceTimestamp)` 计算时间差绝对值
- 按时间差从小到大排序，最接近的排在前面

### 2. 显示优化
- 显示时间差（如"5分钟前"、"2小时前"）
- 显示目标笔记的时间
- 显示内容预览（最多50字符）

### 3. 合并逻辑
- 使用 `Note.mergeNoteAsComment()` 方法执行合并
- 合并后的评论需要保存到数据库
- 源笔记及其所有关联数据需要删除

### 4. 数据一致性
- 使用数据库事务确保操作的原子性
- 合并失败时回滚所有操作

### 5. UI更新
- 合并后刷新列表
- 保持滚动位置
- 显示操作结果提示

## 注意事项

1. **性能考虑**：如果笔记数量很大，可以考虑限制查询范围（如只查询最近N天的笔记）
2. **用户体验**：合并操作可能需要一些时间，使用进度对话框提示用户
3. **错误处理**：需要处理各种异常情况（笔记不存在、数据库错误等）
4. **数据验证**：合并前检查源笔记和目标笔记是否存在
5. **评论ID处理**：合并后的评论ID可能需要重新生成，或者使用源笔记的ID（需要确认数据库设计）

## 扩展功能（可选）

1. **搜索功能**：在合并目标选择对话框中添加搜索框，可以按内容搜索
2. **时间范围筛选**：允许用户选择只显示特定时间范围内的笔记
3. **批量合并**：支持一次选择多个笔记合并到目标笔记
4. **合并预览**：在确认对话框中显示合并后的效果预览
